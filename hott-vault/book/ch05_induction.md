# Chapter 5: Induction / 归纳

归纳类型的一般理论。从具体例子（$\mathbb{N}$, 列表）抽象到 W-type 和初始代数，讨论同伦归纳类型（计算规则放宽为命题等式），最终用 identity systems 刻画恒等类型。

The general theory of inductive types. Abstracts from concrete examples to W-types and initial algebras, discusses homotopy-inductive types (computation rules weakened to propositional equalities), and concludes with identity systems characterizing identity types.

---

## 5.1 Introduction to Inductive Types / 归纳类型导论
[[Inductive_Types/introduction]]

> **归纳类型 = 由构造器自由生成的类型 / Inductive type = type freely generated by constructors.**

归纳类型 $X$ 由有限个构造器自由生成，每个构造器是以 $X$ 为值域的函数（可以有零个参数）。

**例子 / Examples:**
- $\mathbf{2}$: 由 $0_{\mathbf{2}} : \mathbf{2}$ 和 $1_{\mathbf{2}} : \mathbf{2}$ 生成。归纳原理 = 情况分析。
- $\mathbb{N}$: 由 $0 : \mathbb{N}$ 和 $\mathrm{suc} : \mathbb{N} \to \mathbb{N}$ 生成。归纳原理 = 数学归纳法。
- $\mathrm{List}(A)$: 由 $\mathrm{nil}$ 和 $\mathrm{cons} : A \to \mathrm{List}(A) \to \mathrm{List}(A)$ 生成。

**一般模式 / General pattern:**
- **归纳原理 (induction principle):** 证明 $\prod_{x:X} E(x)$，只需对每个构造器给出相应的数据（含递归假设）。
- **计算规则 (computation rules):** 归纳函数在构造器上的行为。
- **递归原理 (recursion principle):** 归纳原理的特例，$E$ 为常值族。

**Theorem (Uniqueness for $\mathbb{N}$).** 设 $f, g : \prod_{x:\mathbb{N}} E(x)$ 满足相同递归方程（命题等式意义下），则 $f = g$。

*证明 / Proof:* 对 $D(x) \defeq f(x) = g(x)$ 做归纳。基础情形用 $f(0) = e_z = g(0)$，归纳步骤用递归方程和归纳假设，最后用函数外延性。

## 5.2 Uniqueness of Inductive Types / 归纳类型的唯一性
[[Inductive_Types/uniqueness]]

> **满足相同归纳原理的类型是等价的（借助 univalence，是相等的）。**
> Types satisfying the same induction principle are equivalent (and by univalence, equal).

**例子 / Example:** 设 $\mathbb{N}'$ 是另一个由 $0' : \mathbb{N}'$ 和 $\mathrm{suc}' : \mathbb{N}' \to \mathbb{N}'$ 生成的类型。利用递归原理定义 $f : \mathbb{N} \to \mathbb{N}'$ 和 $g : \mathbb{N}' \to \mathbb{N}$，由唯一性定理得 $g \circ f \sim \mathrm{id}$ 和 $f \circ g \sim \mathrm{id}$，故 $\mathbb{N} \simeq \mathbb{N}'$。

由 univalence，$\mathbb{N} = \mathbb{N}'$，因此任何关于 $\mathbb{N}$ 的构造和定理自动传递到 $\mathbb{N}'$。

**更一般情形 / More general case:** $\mathrm{List}(\mathbf{1})$ 满足与 $\mathbb{N}$ 相同的归纳原理（将 $\mathrm{nil}$ 对应 $0$，$\mathrm{cons}(\star, -)$ 对应 $\mathrm{suc}$），因此 $\mathrm{List}(\mathbf{1}) = \mathbb{N}$。

## 5.3 W-types / W-类型
[[Inductive_Types/w_types]]

> **W-types = 良基树的类型 / W-types = types of well-founded trees.**

**Definition.** 给定 $A : \mathcal{U}$ 和 $B : A \to \mathcal{U}$，W-type $\mathsf{W}_{a:A} B(a)$ 由以下构造器生成：

$$\mathrm{sup} : \prod_{a:A} \left(B(a) \to \mathsf{W}_{x:A} B(x)\right) \to \mathsf{W}_{x:A} B(x)$$

$A$ 是节点标签的类型，$B(a)$ 是标签 $a$ 的分支数。

**编码自然数 / Encoding natural numbers:**

$$\mathbb{N}^{\mathbf{w}} \defeq \mathsf{W}_{b:\mathbf{2}} \mathrm{rec}_{\mathbf{2}}(\mathcal{U}, \mathbf{0}, \mathbf{1}, b)$$

其中 $\mathbf{0}$ (empty) 对应零参数构造器 $0$，$\mathbf{1}$ (unit) 对应一参数构造器 $\mathrm{suc}$。

$$0^{\mathbf{w}} \defeq \mathrm{sup}(0_{\mathbf{2}}, \lambda x:\mathbf{0}. \mathrm{rec}_{\mathbf{0}}(\mathbb{N}^{\mathbf{w}}, x))$$
$$\mathrm{suc}^{\mathbf{w}} \defeq \lambda n. \mathrm{sup}(1_{\mathbf{2}}, \lambda x:\mathbf{1}. n)$$

**归纳原理 / Induction principle:** 对于 $E : (\mathsf{W}_{x:A} B(x)) \to \mathcal{U}$，要证 $\prod_w E(w)$，只需给出：

$$e : \prod_{a:A} \prod_{f:B(a) \to W} \left(\prod_{b:B(a)} E(f(b))\right) \to E(\mathrm{sup}(a, f))$$

**计算规则 / Computation rule:**

$$\mathrm{rec}(E, e, \mathrm{sup}(a, f)) \jdeq e(a, f, \lambda b. \mathrm{rec}(E, e, f(b)))$$

**Theorem (Uniqueness for W-types).** 满足相同递归方程的两个函数相等。

## 5.4 Inductive Types are Initial Algebras / 归纳类型是初始代数
[[Inductive_Types/initial_algebras]]

> **范畴论视角：归纳类型 = 多项式函子的同伦初始代数。**
> Categorical perspective: inductive types = homotopy-initial algebras for polynomial functors.

**Definition ($\mathbb{N}$-algebra).** $\mathbb{N}$-代数是 $(C, c_0, c_s)$，其中 $C : \mathcal{U}$，$c_0 : C$，$c_s : C \to C$。

$$\mathsf{NAlg} \defeq \sum_{C:\mathcal{U}} C \times (C \to C)$$

**Definition ($\mathbb{N}$-homomorphism).** $(C, c_0, c_s)$ 到 $(D, d_0, d_s)$ 的 $\mathbb{N}$-同态是 $h : C \to D$ 使得 $h(c_0) = d_0$ 且 $\prod_c h(c_s(c)) = d_s(h(c))$。

**Definition (Homotopy-initial).** $\mathbb{N}$-代数 $I$ 是 **h-initial** 如果对每个 $\mathbb{N}$-代数 $C$，$\mathrm{NHom}(I, C)$ 是 contractible。

**Theorem.** $(\mathbb{N}, 0, \mathrm{suc})$ 是 h-initial 的。

**Theorem.** 任何两个 h-initial $\mathbb{N}$-代数相等（不仅仅同构）。

**一般情形 / General case:** 对于 W-type $\mathsf{W}_{a:A} B(a)$，关联的**多项式函子 (polynomial functor)** 为：

$$P(X) = \sum_{a:A} (B(a) \to X)$$

$P$-代数是 $(C, s_C : P(C) \to C)$，即 $(C, \prod_{a:A} (B(a) \to C) \to C)$。

**Theorem.** $(\mathsf{W}_{a:A} B(a), \mathrm{sup})$ 是 h-initial $P$-代数。

## 5.5 Homotopy-Inductive Types / 同伦归纳类型
[[Inductive_Types/homotopy_inductive]]

> **将判断等式 ($\jdeq$) 放宽为命题等式 ($=$)。**
> Weaken judgmental equalities ($\jdeq$) to propositional equalities ($=$).

**动机 / Motivation:** W-type 编码的自然数只能满足**命题**计算规则（不是判断等式）。同伦归纳类型接受这一点，并证明它足够。

**三种等价刻画 / Three equivalent characterizations:**

1. **$\mathsf{W}_d(A,B)$:** 带有依赖消去原理 + 命题计算规则的类型。
2. **$\mathsf{W}_s(A,B)$:** 带有简单递归 + 唯一性 + coherence 的类型。
3. **$\mathsf{W}_h(A,B)$:** 同伦初始 W-代数。

**Theorem.** $\mathsf{W}_d(A,B) \simeq \mathsf{W}_s(A,B) \simeq \mathsf{W}_h(A,B)$，且每个都是 mere proposition。

**Theorem.** 满足 W-type 形成、引入、消去和**命题**计算规则的类型恰好是 h-initial W-代数。

## 5.6 The General Syntax of Inductive Definitions / 归纳定义的一般语法
[[Inductive_Types/general_syntax]]

> **严格正性 (strict positivity) 确保归纳定义的合理性。**

**反例 / Counterexample:** 考虑"构造器" $g : (C \to \mathbb{N}) \to C$。$C$ 出现在箭头左边，导致无法定义递归器，或导致不一致。

**另一反例 / Another counterexample:** 考虑 $k : ((D \to \mathrm{Prop}) \to \mathrm{Prop}) \to D$。虽然 $D$ 出现在协变（covariant）位置，但双重嵌套导致 Cantor 式对角化悖论。

**严格正性条件 / Strict positivity condition:** 在构造器的输入类型中，被定义类型 $W$ 只能出现在：
- 不涉及 $W$ 的类型中，或
- 以 $W$ 为值域的（可能迭代的）函数类型中。

**例子 / Valid example:** $c : (A \to W) \to (B \to C \to W) \to D \to W \to W$。

**递归原理的提取 / Extracting recursion principle:** 对于构造器 $(A \to W) \to (B \to C \to W) \to D \to W \to W$，递归假设将 $W$ 替换为 $P$：

$$(A \to W) \to (A \to P) \to (B \to C \to W) \to (B \to C \to P) \to D \to W \to P \to P$$

## 5.7 Generalizations / 推广
[[Inductive_Types/generalizations]]

- **参数 (Parameters):** 类型如 $A + B$ 和 $\mathrm{List}(A)$ 中 $A, B$ 是参数。每个参数值独立生成一个归纳类型。
- **归纳族 (Inductive families):** $\mathrm{Vec}(n, A)$ 按长度 $n : \mathbb{N}$ 索引。构造器可以改变索引：$\mathrm{cons} : \prod_{n:\mathbb{N}} A \to \mathrm{Vec}(n, A) \to \mathrm{Vec}(\mathrm{suc}(n), A)$。
- **互归纳 (Mutual induction):** 例如 $\mathrm{even}$ 和 $\mathrm{odd}$ 同时定义。
- **归纳-归纳 (Inductive-inductive):** 类型 $A$ 和族 $B : A \to \mathcal{U}$ 同时归纳定义。
- **归纳-递归 (Inductive-recursive):** 类型和从该类型出发的函数同时定义。在同伦视角下尚不清楚如何证明其合理性。

## 5.8 Identity Types and Identity Systems / 恒等类型与恒等系统
[[Identity_and_Paths/identity_systems]]

> **恒等类型是归纳族的特例，identity system 是其抽象刻画。**
> Identity types are a special case of inductive families; identity systems are their abstract characterization.

**恒等类型作为归纳族 / Identity types as inductive families:**
- 两参数版本：$=_A : A \to A \to \mathcal{U}$，构造器 $\mathrm{refl}_a : a =_A a$。归纳原理 = 路径归纳。
- 基于点的版本：固定 $a_0 : A$，$(a_0 =_A \mathord{-}) : A \to \mathcal{U}$，构造器 $\mathrm{refl}_{a_0}$。归纳原理 = 基于路径归纳。

**重要注意 / Important note:** 不是每个路径都等于 $\mathrm{refl}$！恒等类型是**自由生成**的函子，生成元在群胚运算下产生新路径（如 $\mathrm{loop} \cdot \mathrm{loop}$）。

**Definition (Identity system at a point).** 设 $a_0 : A$。一个在 $a_0$ 处的 **identity system** 是有点谓词 $(R, r_0)$（$R : A \to \mathcal{U}$，$r_0 : R(a_0)$）使得对任何 $D : \prod_{b:A} R(b) \to \mathcal{U}$ 和 $d : D(a_0, r_0)$，存在 $f : \prod_{b:A} \prod_{r:R(b)} D(b, r)$ 满足 $f(a_0, r_0) = d$。

**Theorem (Identity systems characterization).** 对于有点谓词 $(R, r_0)$，以下等价：
1. $(R, r_0)$ 是 identity system。
2. 对任何有点谓词 $(S, s_0)$，$\mathrm{ppmap}(R, S)$ 是 contractible。
3. 对任何 $b : A$，$\mathrm{transport}^R(-)(r_0) : (a_0 =_A b) \to R(b)$ 是等价。
4. $\sum_{b:A} R(b)$ 是 contractible。

**Corollary (Equivalence induction).** 给定 $D : \prod_{A,B:\mathcal{U}} (A \simeq B) \to \mathcal{U}$ 和 $d : \prod_{A:\mathcal{U}} D(A, A, \mathrm{id}_A)$，存在 $f : \prod_{A,B:\mathcal{U}} \prod_{e:A \simeq B} D(A, B, e)$ 满足 $f(A, A, \mathrm{id}_A) = d(A)$。

*即：要证关于所有等价的性质，只需证对恒等映射成立。*

**Corollary (Homotopy induction).** 类似地，要证关于所有同伦的性质，只需证对恒等同伦成立。

Inductive types in this chapter only generate "points" — Ch6 will generalize to **higher inductive types (HITs)** that also generate paths and higher paths. The identity system viewpoint provides conceptual foundations for HIT elimination principles.
