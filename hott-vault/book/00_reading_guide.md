# 阅读导引 / Reading Guide

本目录提供逐章的阅读路径，贯穿按概念组织的知识库。每篇章节摘要概括叙事主线，并通过 wikilink 链接到相应的概念页面。

This directory provides a chapter-by-chapter reading path through the concept-organized knowledge base. Each chapter digest summarizes the narrative arc and links to the relevant concept pages.

## 建议路径 / Suggested Paths

**路径一：线性阅读 / Linear**
Ch1 → Ch2 → Ch3 → Ch4 → Ch5 → Ch6 → Ch7 → Ch8 → Ch9 → Ch10 → Ch11

按照原书顺序，适合初次阅读。每章依赖前面的材料。

**路径二：基础优先 / Foundations-first**
Ch1 → Ch2 → Ch4 → Ch3 → Ch7 → Ch5 → Ch6 → Ch8 → Ch9 → Ch10 → Ch11

先建立等价的严格定义（Ch4），再讨论逻辑（Ch3）和 n-type 层级（Ch7），然后进入归纳类型和 HIT。适合有代数拓扑背景的读者。
