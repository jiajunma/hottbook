## 书籍概述 / Book Overview

本书介绍了 **Homotopy Type Theory (HoTT)** —— 一种新的数学基础框架，结合了：
- **类型论 (Type Theory)** 的构造性
- **同伦论 (Homotopy Theory)** 的几何直觉
- **范畴论 (Category Theory)** 的统一视角

> 本书可以被视为一部教科书、研究专著或参考文献。它既可以作为学习 HoTT 的入门教材，也可以作为该领域研究者的参考书。

## 核心主题 / Core Themes

### 1. 类型作为空间 / Types as Spaces
在 HoTT 中，每个类型都可以被看作一个拓扑空间或高维群胚。类型论与同伦论的这种结合是革命性的。

**核心思想 / Key Idea**:
- 类型 $A$ 的元素可以看作空间 $A$ 的点
- 恒等类型 $\id[A]{a}{b}$ 表示从 $a$ 到 $b$ 的路径
- 高阶恒等类型表示路径之间的同伦

### 2. Univalence 公理 / Univalence Axiom
**Univalence axiom** 是 HoTT 最核心的创新：它将类型间的等价等同于身份（相等）。

$$\mathsf{ua} : \prd{A,B:\type} (\eqv{A}{B}) \simeq (A = B)$$

这意味着"相等即等价"——一个数学对象与其等价的所有形式在某种意义上是"相同"的。

### 3. 高阶归纳类型 / Higher Inductive Types
高阶归纳类型允许我们不仅构造点，还构造路径和高维路径。这使得我们可以直接定义如圆 $S^1$ 这样的空间。

## 知识库结构 / Knowledge Base Structure

本知识库按概念领域组织，而非按书的章节顺序：

```
Foundations/              类型论基础（判断、类型构造器）
Identity_and_Paths/       恒等类型与路径（路径运算、transport、encode-decode）
Equivalences/             等价（三种定义、闭合性质、object classifier）
Axioms/                   公理（Univalence、函数外延性、选择公理）
Truncation_Levels/        截断层级（可缩、命题、集合、n-type、模态）
Logic/                    逻辑（命题截断、经典 vs 直觉主义）
Inductive_Types/          归纳类型（W-type、初始代数、一般语法）
Higher_Inductive_Types/   高阶归纳类型（圆、球面、pushout、商、截断）
Homotopy_Theory/          同伦论（π₁(S¹)、Hopf、Freudenthal、van Kampen）
Category_Theory/          范畴论（预范畴、Yoneda、Rezk completion）
Set_Theory/               集合论（基数、序数、累积层级 V）
Real_Numbers/             实数（Dedekind、Cauchy、surreal）
```

每个领域有一个 **00_MOC.md**（Map of Content）作为导航入口。
