# 逻辑 / Logic in HoTT

## 概述 / Overview

朴素的命题即类型使得选择公理平凡成立但排中律失败。将逻辑限制为纯命题可恢复传统逻辑。命题截断 $\|A\|$ 连接两者。"Merely" = 截断后的。

Naive propositions-as-types makes AC trivially true but LEM false. Restricting to mere propositions recovers conventional logic. Propositional truncation $\|A\|$ bridges the gap. "Merely" = truncated.
