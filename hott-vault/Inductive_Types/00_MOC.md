# 归纳类型 / Inductive Types

## 概述 / Overview

归纳类型的特征是作为多项式函子的初始代数。W-类型提供通用编码。同伦归纳类型使用命题计算规则。推广包括归纳-归纳类型和归纳-递归类型。

Inductive types are characterized by being initial algebras of polynomial functors. W-types provide a universal encoding. Homotopy-inductive types use propositional computation rules. Generalizations include inductive-inductive and inductive-recursive types.
