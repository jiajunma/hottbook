# 自然数类型 / The Natural Number Type

## 定义 / Definition

自然数 $\mathbb{N}$ 是类型论中第一个真正的**归纳类型 (inductive type)**——由构造子递归地生成。
$\mathbb{N}$ is the first real **inductive type** in type theory — recursively generated by constructors.

构造子 / Constructors:

$$0 : \mathbb{N} \qquad \mathsf{succ} : \mathbb{N} \to \mathbb{N}$$

所有自然数由这两个构造子穷尽生成：$0,\; \mathsf{succ}(0),\; \mathsf{succ}(\mathsf{succ}(0)),\; \ldots$

## Lean 4

```lean
-- 从零定义 Nat
inductive MyNat where
  | zero : MyNat
  | succ : MyNat → MyNat

open MyNat

def myAdd : MyNat → MyNat → MyNat
  | zero,   n => n
  | succ m, n => succ (myAdd m n)

-- 归纳证明
theorem myAdd_zero (n : MyNat) : myAdd n zero = n := by
  induction n with
  | zero => rfl
  | succ n ih => simp [myAdd, ih]
```

## 编程视角 / Programming Perspective

### 自然数就是递归数据结构 / Nat is a recursive data type

```python
# Haskell: data Nat = Zero | Succ Nat
# Python 近似：
from __future__ import annotations
from dataclasses import dataclass

class Zero:
    """0 : ℕ"""
    pass

@dataclass
class Succ:
    """succ : ℕ → ℕ"""
    pred: Zero | Succ

Nat = Zero | Succ

# 具体的数 / Concrete numbers
zero  = Zero()                    # 0
one   = Succ(Zero())              # succ(0) = 1
two   = Succ(Succ(Zero()))        # succ(succ(0)) = 2
three = Succ(Succ(Succ(Zero())))  # succ(succ(succ(0))) = 3
```

```typescript
// TypeScript: 递归类型定义
type Nat = { tag: "zero" } | { tag: "succ"; pred: Nat };
const zero: Nat = { tag: "zero" };
const succ = (n: Nat): Nat => ({ tag: "succ", pred: n });

// 0, 1, 2, 3
const one = succ(zero);
const two = succ(one);
const three = succ(two);
```

## 递归子 / Recursor (Primitive Recursion)

递归子是**对自然数编程**的基本工具——它就是 `fold`：
The recursor is the fundamental tool for **programming** with natural numbers — it IS `fold`:

$$\mathsf{rec}_{\mathbb{N}} : \prod_{(C:\mathcal{U})} C \to (\mathbb{N} \to C \to C) \to \mathbb{N} \to C$$

计算规则 / Computation rules:
- $\mathsf{rec}_{\mathbb{N}}(C, c_0, c_s, 0) \equiv c_0$
- $\mathsf{rec}_{\mathbb{N}}(C, c_0, c_s, \mathsf{succ}(n)) \equiv c_s(n,\; \mathsf{rec}_{\mathbb{N}}(C, c_0, c_s, n))$

翻译成编程语言：$c_0$ 是基础情况，$c_s$ 接受"当前数"和"递归结果"。
In programming terms: $c_0$ is the base case, $c_s$ takes "current number" and "recursive result".

```python
def rec_nat(c0, cs, n):
    """rec_ℕ(C, c₀, cₛ, n) — 原始递归/fold"""
    if isinstance(n, Zero):
        return c0
    else:
        return cs(n.pred, rec_nat(c0, cs, n.pred))
```

### 具体例子：加法 / Concrete Example: Addition

$$\mathsf{add}(0, n) :\equiv n \qquad \mathsf{add}(\mathsf{succ}(m), n) :\equiv \mathsf{succ}(\mathsf{add}(m, n))$$

逐步计算 $\mathsf{add}(2, 3)$ / Step-by-step trace:

$$\mathsf{add}(\mathsf{succ}(\mathsf{succ}(0)),\; 3) \equiv \mathsf{succ}(\mathsf{add}(\mathsf{succ}(0),\; 3)) \equiv \mathsf{succ}(\mathsf{succ}(\mathsf{add}(0,\; 3))) \equiv \mathsf{succ}(\mathsf{succ}(3)) \equiv 5$$

```typescript
function add(m: Nat, n: Nat): Nat {
    if (m.tag === "zero") return n;          // add(0, n) ≡ n
    return succ(add(m.pred, n));              // add(succ(m), n) ≡ succ(add(m, n))
}
// add(two, three) → succ(add(one, three)) → succ(succ(add(zero, three)))
//                 → succ(succ(three)) = 5 ✓
```

### 更多例子 / More Examples

**双倍 / Double:** $\mathsf{double}(0) :\equiv 0$，$\mathsf{double}(\mathsf{succ}(n)) :\equiv \mathsf{succ}(\mathsf{succ}(\mathsf{double}(n)))$

**乘法 / Multiplication:** $\mathsf{mult}(0, n) :\equiv 0$，$\mathsf{mult}(\mathsf{succ}(m), n) :\equiv \mathsf{add}(n, \mathsf{mult}(m, n))$

## 归纳原理 / Induction Principle

归纳原理是递归子的**依赖版本**——它不是计算值，而是**证明性质**。
The induction principle is the **dependent version** of the recursor — it doesn't compute values, it **proves properties**.

$$\mathsf{ind}_{\mathbb{N}} : \prod_{(C:\mathbb{N} \to \mathcal{U})} C(0) \to \left(\prod_{(n:\mathbb{N})} C(n) \to C(\mathsf{succ}(n))\right) \to \prod_{(n:\mathbb{N})} C(n)$$

这正是**数学归纳法** / This IS mathematical induction:

| 归纳法步骤 / Induction step | 类型论对应 / Type theory | 编程类比 / Programming |
|---|---|---|
| 基础：证明 $P(0)$ | 给出 $c_0 : C(0)$ | base case |
| 递推：$P(n) \Rightarrow P(n+1)$ | 给出 $c_s : \prod_{(n:\mathbb{N})} C(n) \to C(\mathsf{succ}(n))$ | recursive case |
| 结论：$\forall n.\; P(n)$ | 得到 $\prod_{(n:\mathbb{N})} C(n)$ | total function |

> **关键洞察 / Key Insight**: 递归子计算**值**（$C$ 是常量类型），归纳原理证明**性质**（$C(n)$ 是依赖于 $n$ 的命题）。数学归纳法是归纳原理的特殊情况！
> The recursor computes **values** ($C$ is a constant type); the induction principle proves **properties** ($C(n)$ is a proposition depending on $n$). Mathematical induction is a special case!

### 归纳证明示例 / Induction Proof Example

**加法结合律 / Associativity of addition:**

$$\mathsf{assoc} : \prod_{(i,j,k:\mathbb{N})} \mathsf{add}(i,\, \mathsf{add}(j,k)) =_{\mathbb{N}} \mathsf{add}(\mathsf{add}(i,j),\, k)$$

对 $i$ 归纳 / By induction on $i$:
- **基础 / Base**: $\mathsf{add}(0, \mathsf{add}(j,k)) \equiv \mathsf{add}(j,k) \equiv \mathsf{add}(\mathsf{add}(0,j), k)$，由 $\mathsf{refl}$ 见证
- **递推 / Step**: 假设 $\mathsf{assoc}(i,j,k)$ 成立，对两边应用 $\mathsf{succ}$ 得到 $\mathsf{assoc}(\mathsf{succ}(i), j, k)$

这和你在离散数学课上学的归纳法完全一样——只是这里"证明"是一个**程序**。
This is exactly the induction you learned in discrete math — except here a "proof" is a **program**.
