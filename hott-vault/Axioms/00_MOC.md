# 公理 / Axioms

## 概述 / Overview

HoTT 在 Martin-Lof 类型论基础上增加了两个关键公理：Univalence $(A =_\mathcal{U} B) \simeq (A \simeq B)$ 和函数外延性 (可由 UA 推出)。选择公理是可选的，且仅限于集合。

HoTT adds two key axioms to Martin-Lof type theory: univalence $(A =_\mathcal{U} B) \simeq (A \simeq B)$ and function extensionality (which follows from UA). The axiom of choice is optional and restricted to sets.
