# 集合论 / Set Theory

## 概述 / Overview

集合 (0-类型) 构成一个 $\Pi W$-预拓扑。基数、序数和累积层叠 $V$ 都可以在 HoTT 中定义。累积层叠 $V$ 是一个 HIT。

Sets (0-types) form a $\Pi W$-pretopos. Cardinals, ordinals, and the cumulative hierarchy $V$ can all be defined within HoTT. The cumulative hierarchy $V$ is a HIT.
