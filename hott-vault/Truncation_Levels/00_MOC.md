# 截断层级 (n-types) / Truncation Levels

## 概述 / Overview

类型按同伦层级分层：可缩 (-2) $\subset$ 纯命题 (-1) $\subset$ 集合 (0) $\subset$ 1-类型 $\subset$ ...。$A$ 是 $n$-类型当且仅当其所有恒等类型是 $(n-1)$-类型。截断 $\|A\|_n$ 自由地将任意类型变为 $n$-类型。

Types are stratified by homotopy level: contractible (-2) $\subset$ mere proposition (-1) $\subset$ set (0) $\subset$ 1-type $\subset$ ... $A$ is an $n$-type iff all its identity types are $(n-1)$-types. Truncation $\|A\|_n$ freely makes any type into an $n$-type.
