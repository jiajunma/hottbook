# 同伦论 / Homotopy Theory

## 概述 / Overview

HoTT 中的合成同伦论。关键结果：$\pi_1(S^1) \cong \mathbb{Z}$ (编码-解码)、Hopf 纤维化、Freudenthal 悬挂定理、van Kampen、Whitehead、Blakers-Massey。全部在类型论内部证明，无需点集拓扑。

Synthetic homotopy theory in HoTT. Key results: $\pi_1(S^1) \cong \mathbb{Z}$ (encode-decode), Hopf fibration, Freudenthal suspension theorem, van Kampen, Whitehead, Blakers-Massey. All proved internally without point-set topology.
